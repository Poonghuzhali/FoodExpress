🍔 FoodExpress
GitHub Live App

Delicious food, delivered fast

Full-stack food delivery platform built with Django, HTML, CSS, and JavaScript. Order from restaurants, manage menus, track deliveries, and run the entire system from one app.

Open Live App → View Source Code
Note: This GitHub Pages site is the project homepage only. The Django application runs on Render because GitHub Pages cannot host Python/Django backends.

Features

👤 Customer

Browse restaurants, cart, checkout, payments, order tracking, reviews & coupons.

🏪 Restaurant

Manage profile, menu, accept orders, update status, and view sales reports.

🛵 Delivery

Accept deliveries, update status, and track earnings.

⚙️ Admin

Manage users, restaurants, orders, categories, coupons, and analytics.
